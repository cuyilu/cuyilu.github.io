园（收）

发表于 2017-11-02 | 分类于 -日志

信

园：
　　展信如见，虽好久不见。
　　你说想看一下我更新博客，就在博客这里给你写封信吧，放心，我加了密，希望不会有人无聊的去破解吧，不过也没什么，至少我们共同认识的人不会这么做。
　　每次要写点什么，脑袋就一片空白了，哈哈，我都不知道我要搭建这个博客干什么，感觉都写不出东西出来。
　　我叫你园可以吧，我希望这样做不会引起你有任何不舒服的感觉，我最近有种感觉，别人不开心我会很容易察觉，所以你别不开心，不然可能我为突然而来的心闷心痛而烦恼。
　　聊聊开心的事，一直以来都是，最开心的就是可以有个人无所不谈，哈哈，那样的时候就是忘记自己现在的时间和状态，专注做一件事，和你聊天也是这样的。
　　学了点心理学，感觉整个人轻松了一点，因为知道了自己的状态还没那么糟糕，至少开始懂得去了解自己的思考方式和个性变化，其实很多时候人都是害怕一些未知或者不存在的东西，当你明白原来那个东西长那样，其实也没那么可怕。
　　有一种疗法叫认知行为疗法，说复杂一点就是一步一步逼近事实，一步一步拆穿不合适甚至不正确的原因，再去一点一点做出另外的行为适应新的原因。说简单一点就是要暴露，藏太久就会有毛病。我可能现在只告诉你一个，但是已经能舒坦很多，你也有“亲密”的朋友，我相信你知道我在说什么。
　　当然，这样说好像把你说成了垃圾桶，我是想说很感谢你能当起这个可以在我跌倒摔得不知所措的时候还能过来帮我拍拍灰的可爱人，说了一句：“我也跌倒过，没人帮不好受”，其实我也一直很乐意当这样的人，只是我也需要有人这么做吧。
　　我真的不会写东西，写着写着又哑了，就像我跟你讲电话一样，不过我倒是更乐意听你说多一点，有时候听你说着说着会笑什么的，那种感觉比什么都好。↙
　　好了，说点别的，我怕那样这样说下去是不是会太肉麻了，但只是心里想什么就说点什么吧，可能这一点美国人做得好一点，至少他们宣传的诚实是做人之本并实践地挺好，我们也在宣传诚信，但由于东方人偏内敛性格的原因，反而更容易说出和做出不符合自己真实感觉的事情。每当这样做一次，人就需要疗养一次，压力也是如此。疗养不来了，就只能躲，躲久了就会怀疑自己的价值，怀疑了自己价值之后再想回来总觉得有堵无形的墙在面前挡着，对面的人在呼喊，自己还在想这堵墙为什么存在，于是就做出我回不去的姿态，对面的人也不再喊了，自己不知觉的也把这堵墙砌好了。说出来是有点悲哀的样子，但是这样状态或许才是每个人应该有的状态。总有一天自己会让一个彻彻底底信得过的人过墙这边来，那个人可能会帮你开扇门，其描述来自的世界会链接你的世界，并渐渐让你接纳链接的更大的世界。我也希望你能相信这一点，真的。
　　乱说一通说了这些，希望不会给你带来不好的感觉，有的话记得也要告诉我。还有哦，如果有可能的话，我希望能和你一起出去找点喜欢的事情做或者看点美的东西，至少我心里是这么想的。或许这就是想“ask someone out”的感觉，哈哈。
　　说到这里吧，有机会我会继续写点其他东西，有什么好建议记得告诉我。
　　　
　　愿好！！
